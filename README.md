# 致谢 | [W3C中文翻译贡献者名录](https://github.com/w3c-beihang/translation-contribution/blob/main/list.md)

### 衷心感谢所有参与W3C技术标准与文档中文翻译工作的贡献者们。正是你们的热忱、专业与无私奉献，为Web中文社区提供了便捷的参考资料。[本页面](https://github.com/w3c-beihang/translation-contribution/blob/main/list.md)旨在汇总和展示这些宝贵的翻译成果，并记录每一位贡献者的付出。

#### W3C的工作语言为英语，其所有文档的官方版本均以英语形式发布于W3C官网。为促进全球产业发展与技术传播，我们诚邀各方将W3C标准文件翻译为多语言版本，以服务更广泛的社区。

#### 翻译工作需投入大量专业精力，我们由衷欢迎您加入翻译贡献者行列，协助推进W3C标准规范在各语言中的本地化与普及。如您有意参与，欢迎进一步了解W3C的[翻译政策与具体协作流程](https://www.w3.org/2025/10/legacy-cn-website/translation.html)。
